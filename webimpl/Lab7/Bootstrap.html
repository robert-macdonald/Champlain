Twitter Bootstrap

  • front-end framework that helps developers to jump start the web development process
  • can concentrate on HTML and let Bootstrap take care of the CSS and JS
  • features include nav bars, image sliders, styled buttons, dropdown menus, ...
  • grid system for MFD (mobile first design), does RWD for you

To use:

download Bootstrap
  • put the Bootstrap folders inside your web site folder
  • set your files up to be MFD (<meta name="viewport" content="width=device-width, initial-scale=1">)
  • link to the css file (<link href="css/bootstrap.min.css" rel="stylesheet">)
  • attach the script file (<script src="js/bootstrap.min.js"></script>)

OR if you are going to be connected to the Internet at all times you can use the CDN (Content Delivery Network) method

For testing/debugging purposes:

  • link to the non-minimized CSS and JavaScript files - you are able to look at code to find the options that you need
  • link to the css file (<link href="css/bootstrap.css" rel="stylesheet">)
  • attach the script file (<script src="js/bootstrap.js"></script>)
